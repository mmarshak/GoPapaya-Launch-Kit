  • Step 1: Get the Code

    • CocoaPods
    • Carthage
    • Manual

    Our SDK is available through CocoaPods. To install it, simply add the following line to your Podfile:

    pod "LaunchKit"

    LaunchKit is also available through Carthage. Add the following line to your Cartfile: 

    github "LaunchKit/launchkit-ios"

    You can install the LaunchKit SDK manually by downloading the latest release (.zip), and copy the files from LaunchKit/Classes into your project.

    Download the Latest Release
